## Usage

Run the project directly:

    $ clojure -m {{name}}.main

Run the project's tests (they'll fail until you edit them):

    $ clojure -A:test:runner

Build an uberjar:

    $ clojure -A:uberjar

Run that uberjar:

    $ java -jar logger.jar
